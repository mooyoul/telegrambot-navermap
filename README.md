## Requirements

* Node.js >= 4
* MongoDB >= 3
* Telegram Bot API Key

## Getting Started
```bash
$ git clone https://github.com/mooyoul/telegrambot-navermap.git
$ cd telegrambot-navermap
$ npm install
$ cp .env.example .env
$ vi .env # You must edit configurations!
```

### Running server
```bash
$ node server.js
```
